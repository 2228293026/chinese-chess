# 快速开始指南

## 📦 项目已完成重构

原单文件 `Main.html` 已拆分为模块化 TypeScript 项目，所有新功能已实现。

## 🚀 立即运行

### 方式1: 开发服务器（推荐）
```bash
cd "D:\Projects\中国象棋"
npm run dev
```
浏览器自动打开: http://localhost:3002/

### 方式2: 直接打开
双击 `dist/Main.html` 即可（需要现代浏览器支持 ES6 模块）

### 方式3: 生产构建
```bash
npm run build
```
产物在 `dist/` 目录

## 🎮 操作说明

- **走子**: 点击己方棋子选中，再点击目标格
- **切换选中**: 选中状态下点击另一己方棋子
- **自动对弈**: 点击"▶ 自动"按钮
- **悔棋**: 点击"↩️ 悔棋"（最多50步）
- **编辑模式**: 点击"✎ 编辑"，自由摆放残局
- **日志**: 点击"📋 日志"查看AI思考过程
- **主题**: 点击"主题"按钮循环切换
