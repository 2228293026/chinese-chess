# 中国象棋 - 测试清单

## 启动游戏

```bash
cd "D:\Projects\中国象棋"
npm run dev
```

然后在浏览器中打开: http://localhost:3002/

## 功能测试清单

### 1. AI难度调节 ✅
- [ ] 打开页面，默认难度为"普通 (深度3)"
- [ ] 拖动滑动条到"新手"（1），AI走子变慢/水平下降
- [ ] 拖动到"困难"（4），AI思考变深，走子更强
- [ ] 状态栏显示当前难度："AI难度: 新手 (深度1)"等

### 2. 着法提示 ✅
- [ ] 点击红方棋子（如炮）
- [ ] 看到绿色圆圈显示所有可移动位置
- [ ] 切换选中不同棋子，提示位置正确更新

### 3. 威胁显示 ✅
- [ ] 进行对局，当一方被将军时，状态显示"⚠️ 被将军"
- [ ] 对方棋子可能将军的线路显示红色边框
- [ ] 红色框准确标出威胁位置

### 4. 棋子位置表 ✅
- [ ] 开局走子随机（前4步）
- [ ] 中局AI会根据棋子位置加分做出合理决策
- [ ] 马在中心加分，车在开阔线加分

### 5. 终局胜负标注 ✅
- [ ] 对局结束（将死），状态显示"🏆 第X步 · XX将死XX方！"
- [ ] 显示具体步数和致胜棋子

### 6. 配色方案切换 ✅
- [ ] 点击"主题"按钮
- [ ] 背景色切换（绿色/紫色/深蓝/棕色）
- [ ] 按钮名循环显示："主题: 经典" → "现代" → "暗色" → "木纹"

### 7. 双方胜率显示 ✅
- [ ] 开局时胜率接近50%/50%
- [ ] 吃子后，优势方胜率上升
- [ ] 数字实时更新（如"红: 62% | 黑: 38%"）
- [ ] 游戏结束时显示"红胜 100%"或"黑胜 100%"

### 8. AI思考日志 ✅
- [ ] 开启"日志"面板
- [ ] 每步AI走子后，日志显示步数、颜色、着法、评估分
- [ ] 点击"清空"按钮清空日志

### 9. 其他功能 ✅
- [ ] 点击"自动"，AI自动对弈
- [ ] 点击"悔棋"，回退一步
- [ ] 点击"新局"，重置棋盘
- [ ] 编辑模式：点击"编辑"，放置/拾取棋子
- [ ] 音效开关：点击"音效"按钮可静音/开启

### 10. 响应式与动画 ✅
- [ ] 棋子移动有平滑动画
- [ ] 点击有反应，无卡顿
- [ ] 界面在不同尺寸窗口下正常显示
